Student Record Application

As a final project for one of the early Computer Science courses, we created a student record application. The requirements for this application included being able to allow the user to create and add a new student record, remove a student record, access any and all existing student records, and search for a student record by name or student ID. When adding courses to a student's records, the user could enter the course, number of credits the course was worth, along with the grade that the student got. Then based off of all the grades that one student has, the system could calculate the student's GPA. The application was created in Java.

Wikipedia Data Searcher

For this project, one partner and I were to create a software application that pulls live data from Wikipedia. The data we were specifically looking for was the most recent editors along with the most active editors of an article on Wikipedia. In the first iteration, the main focus was to get the software to where it can pull and display the data. In the second iteration, we were also tasked with creating a graphical user interface that can display the results. The application was created in Java and also used Java FX.


Restraunt Order

For a final project in an advanced programming class, students were organized into groups of four where we had to create our own non-trivial piece of software. The project took place over three iterations, and we were to have some executable software at each release. My group decided to do a restraunt order system where a customer could select what food they wanted to order, and as they selected what they would want, the software would add up the prices until the customer was ready to check out. Then not only does it display their total, but it will give the option to add a tip, and if the customer enter's an amount (could be a percentage or specific amount), it will recalculate a final total with the tip added. The software was created using Java and Java FX.
